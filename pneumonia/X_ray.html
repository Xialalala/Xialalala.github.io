Pneumonia Detection with deep CNN model¶

Machine learning and Deep learning have a huge implementation in healthcare but applying them in healthcare isn't that simple. If we are able to apply machine learning in healthcare field very well, it will largely benefit the world and the way we living.

In this project, we will see how deep learning helps classify x-ray images for helping doctors detecting Pneumonia patients. Pneumonia is a very common disease, which can be caused by a variety of organisms including bacteria, viruses and fungi. The recently pandemic disease COVID-19 showed the similar pattern. And before the production of virus detection kit, doctors have to reply on the X-ray picture to determine whether the patients suffered from COVID-19. Thus, it is very helpful to develop a robust deep learning model.

Data augmentation¶

In order to avoid overfitting problem, I need to expand artifically the dataset and make it even larger. Here I didn't add rotation because it is medical image, the orientation should always be vertical. I make the following changes:

  1. Randomly zoom by 20%
  2. Randomly shift the image horizontally and vertically by 10% of the width and height
  3. Rescale image to range 0-1
  4. Resize the image to 128*128 with same size
  5. Set the color mode as grayscale

In [19]:
model = Sequential()

input_shape = (128, 128, 1)

model.add(Conv2D(16, kernel_size=(3, 3),
                 activation='relu',
                 input_shape=input_shape))
# 32 3x3 kernels
model.add(Conv2D(32, (3, 3), activation='relu'))

# Reduce by taking the max of each 2x2 block
model.add(MaxPooling2D(pool_size=(2, 2)))

# Dropout to avoid overfitting
model.add(Dropout(0.20))

# Flatten the results to one dimension for passing into our final layer
model.add(Flatten())

# A hidden layer to learn with
model.add(Dense(16, activation='relu'))

# Final categorization with 0 for Normal and 1 for Pneumonia
model.add(Dense(1, activation='sigmoid'))

VGG16 transfer learning¶

VGG16 is a convolutional neural network model proposed by K. Simonyan and A. Zisserman from the University of Oxford in the paper Very Deep Convolutional Networks for Large-Scale Image Recognition. The model achieves 92.7% top-5 test accuracy in ImageNet, which is a dataset of over 14 million images belonging to 1000 classes. It was one of the famous model submitted to ILSVRC-2014.

def Vgg_16_model (n_row, n_column, n_channel, num_classes= None):
  vgg = VGG16(input_shape= [n_row,n_column, n_channel], weights='imagenet', include_top=False)
  for layer in vgg.layers:
    layer.trainable = False
  flatten1 = Flatten()(vgg.output)
  dense1 = Dense(256, activation="relu")(flatten1)
  dropout1 = Dropout(0.5)(dense1)
  prediction = Dense(num_classes, activation="softmax")(dropout1)

  model = Model(inputs=vgg.input, outputs=prediction)
  print(model.summary)
  model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
  return model

Summary¶

In this project, I found that Augmentation helps a lot in reducing overfitting and improve model performance. Because I have trained the first model when there is no augmentation added, the accuracy was 70%. After image augmentation, the accuracy improved from 70% to 88%.

The modified transfering VGG16 CNN model improved the accuracy by 2% and it saved a lot of time comparing building the first model from raw.

The final accuracy in the project reached 90%, I think there is room to further improve the accuracy. However, by looking at the confusion matrix, the error is still large. Thus, I think machine learning can only be working as a reference when doctor making decisions.

The similar modeling method is also applicable to the diagnosis of COVID-19. There are people who kindly uploaded the COVID-19 x-ray image data link to the data source
